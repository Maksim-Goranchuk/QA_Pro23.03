Инструмент для QA: вставляете описание фичи или требований — получаете готовый чеклист тест-кейсов, разбитых на три группы.
Одностраничное веб-приложение, которое использует Gemini API для автоматической генерации тест-кейсов по текстовому описанию. На выходе — структурированный список проверок:

Позитивные — happy path и его вариации, когда всё работает как ожидается
Негативные — ошибочный ввод, нарушение бизнес-правил, ошибки системы
Граничные значения — min/max длины, пустые поля, спецсимволы, нулевые значения

Быстрый старт
1. Получите API-ключ Gemini
Зарегистрируйтесь на Google AI Studio и создайте ключ.
2. Настройте конфиг
Создайте файл config.js в корне проекта:
jsexport const GEMINI_API_KEY = 'ваш_ключ_сюда';
3. Откройте в браузере
Установите расширение Live Server для VS Code, затем откройте index.html и нажмите «Go Live» в правом нижнем углу редактора. Приложение откроется в браузере автоматически.

Структура проекта
├── index.html   — разметка
├── style.css    — стили (светлая/тёмная тема)
├── app.js       — логика и работа с API
└── config.js    — API-ключ 

Использование:
Вставьте описание фичи в текстовое поле — чем подробнее, тем точнее результат
Нажмите «Сгенерировать проверки» или используйте Ctrl+Enter / Cmd+Enter
Получите чеклист — он появится ниже, сгруппированный по типам проверок
